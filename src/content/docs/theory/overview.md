---
title: Przegląd User Harness
description: Czym jest User Harness i jak działa rekonstrukcja ludzkiego umysłu
---

User Harness to framework opracowany przez University of Illinois Urbana-Champaign (maj 2026), który modeluje ludzki proces poznawczy jako sekwencję:

1. **Obserwacja** — człowiek odbiera tylko część rzeczywistości (filtr epistemiczny Ω)
2. **Przekonanie** — na podstawie obserwacji aktualizuje swoje przekonania (γ)
3. **Decyzja** — przekonania + cele → akcja (π)
4. **Akcja** — zmiana środowiska

Aplikacja **Mind Harness** odwraca tę architekturę: zamiast manipulacji — samoświadomość.
